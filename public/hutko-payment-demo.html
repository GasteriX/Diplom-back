Hutko JavaScript SDK (тест)

Кроки з документації Hutko: створіть замовлення в API → отримайте token → оплатіть карткою нижче.

Лише корінь сервера, без /api і без Swagger (#).

Сторінку відкрито як локальний файл (file://). Браузер блокує запити до API. Відкрийте демо через сервер: http://localhost:3000/demo/hutko-payment-demo.html
